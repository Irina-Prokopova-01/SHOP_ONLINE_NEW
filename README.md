## ONLINE SHOP 

#Описание:
Проект он-лайн магазина. Реализован функциональный вид следующих страниц:

* главная;
* контакты;
* blog

В проекте разрабатывается приложение "catalog", "blog".

# Установка:

* Клонируйте репозиторий:
https://github.com/Irina-Prokopova-01/SHOP_ONLINE_NEW
